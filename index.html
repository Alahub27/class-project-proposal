Project Name: Task Tracker

Last Updated: 19 November 2023

Team Members: Alanna, Juan, Alan, Angelo

Application Overview:

In the application the user is going to immediately see the listed tasks they have added
If it was not done, then the task would be the color red and it would show the percentage of how much the assignment was done.
There would also be an option for what kind of formatting the user would like, they could choose if they want it in a list style or in a calender style.

Technology Overview:

Language:

The languages I will be incorporating in my task tracker project are CSS, Javascript, and HTML.

Libraries:

The library would be used to get data to include a calender feature.

Platform:

Windows

Hosting:

Github

Features List: On Time

  • To add or delete tasks.
  • Unfinished tasks would be the color red and finished tasks would be green.
  • The finished task would be crossed out once done.
  • There will be a drop down button where there is a list of categories created by the user each task belongs in.
  • The dates each assignment is due and how the list goes from the closest date to the farthest date the homework is due.
  • Allow the user to add comments or notes.

Advanced Features List: Given more time

  • Changing of format of tasks in a list or in a calender.
  • Include the percentage of how much the assignment is finished.
  • Give users more options of formatting, so they can change the formatting of the list of task or color to anything they would like.

Basic Technical Features:

  • To have easier accessibility of the tasks and create an organizaed interface for the user.
  • The content of the tasks would be displayed immediately as soon as the user opens it, in an organized listor whatever format the user chooses.
  • The dates would be listed along with the assignments.

Advanced Technical Features:

  • Include tracking the duration of the users worktime when completing assignments.
  • The percentage of how much has been done of the task.
  • I would like to include a search bar for users to search up a specific task they need, it would be less time consuming.
